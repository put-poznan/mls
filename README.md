# My LS

Program that is simple `ls` implementation.

## Requirements

- Any C compiler that accept valid C89 code
- \*NIX system

## Compilation and run

    make
    ./bin/mls

## Options

- `-a` - do not skip files that begin with dot
- `-l` - long format
- `-r` - print in reverse alphabetic order
